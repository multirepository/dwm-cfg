dwm configuration by multirepository

```dwm version 6.8```

Patches:
1. alwayscenter - https://dwm.suckless.org/patches/alwayscenter/
2. awesomebar - https://dwm.suckless.org/patches/awesomebar/
3. Cool autostart - https://dwm.suckless.org/patches/cool_autostart/
4. fullgaps - https://dwm.suckless.org/patches/fullgaps/
5. fullscreen - https://dwm.suckless.org/patches/fullscreen/
6. systray - https://dwm.suckless.org/patches/systray/

Build dependencies:
```build-essential libx11-dev libxft-dev libxinerama-dev make xinit xserver-xorg```

Configuration Dependencies:
1. ```alacritty```
2. ```dbus-x11```
3. ```feh```
4. ```flameshot```
5. ```font-awesome```
6. ```qt5ct```
7. ```xdg-desktop-portal```
8. ```xdg-desktop-portal-gtk```

Build:
1. ```cd ~ && git clone https://github.com/multirepository/dwm-cfg```
2. ```cd ~/dwm-cfg/ && chmod +x build.sh && ./build.sh```
